# Security

## Intended Users
This repository is intended for the course professor, TA, and the student author of the project. It contains coursework for CSE 3000 and is not intended for production use.

## Security Risk Assessment
The repository contains Python and coursework-related code. If accessed by unintended users, the main risk would be academic integrity concerns, such as copying project work. No sensitive personal data or credentials are stored in this repository.

## Repository Security Measures
The repository uses GitHub branch protection rules. Pull requests require code owner review before merging, which helps prevent unauthorized changes. Direct branch deletion and non–fast-forward pushes are also restricted.

Because the repository only contains coursework and no sensitive data, these measures are sufficient.
